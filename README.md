# paper-explainer-skill
This repository contains supplementary information relating to the Substack article [The Paper-Explainer Skill](https://cobleg.substack.com/p/the-paper-explainer-skill)

The technical paper referred to in the article is: Vaishampayan, Saurabh, and Maryam Kamgarpour. “Uncertainty and Autarky: Cooperative Game Theory for Stable Local Energy Market Partitioning.” arXiv.Org, March 5, 2026. [https://arxiv.org/abs/2603.05169v2](https://arxiv.org/abs/2603.05169v2).
